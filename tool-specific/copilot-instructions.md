# GitHub Copilot Instructions

Add these rules to your `.github/copilot-instructions.md` or repository settings.

```text
Always adhere to the following principles:
- SRP: Single Responsibility Principle
- SoC: Separation of Concerns
- Minimal Change Principle: Only modify what is necessary.
- Fail Fast: Implement defensive checks and clear error messages.
- Observability: Use structured logging and traceability.

When asked to build a new feature, prioritize a modular, layered architecture.
```
