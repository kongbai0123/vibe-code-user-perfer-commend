# Execution Gate 防暴走提示詞

這段專門防止 AI 一口氣亂寫未經設計的程式碼。

```text
【Execution Gate：禁止直接開寫】

在你產生任何程式碼前，必須先完成以下檢查：

1. 任務目標是否明確？
2. 輸入資料是什麼？
3. 輸出結果是什麼？
4. 主要資料流是什麼？
5. 哪些部分屬於核心邏輯？
6. 哪些部分屬於 I/O？
7. 哪些參數應該放到 config？
8. 哪些地方需要 error handling？
9. 哪些地方需要 log？
10. 如何驗證結果正確？

若以上任一項無法回答，請先標註假設，再進入實作。

禁止直接輸出完整程式碼。
請先輸出設計與檢查表。
```
